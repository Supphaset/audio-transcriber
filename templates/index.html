💡 How to Use

  1. Select your audio files - Choose one or more audio files (M4A, MP3, WAV, MP4)
  2. Choose AI model - Select transcription model (GPT-4o Mini recommended)
  3. Pick output format - Choose text, subtitles (SRT/VTT), or detailed JSON
  4. Add guidance (optional) - Provide terminology hints for better accuracy
  5. Choose languages - Select audio language and preferred summary language
  6. Test mode (optional) - Enable to process only 1 minute for quick testing
  7. Click "Start Transcription" - Wait for processing to complete
  8. Download results - Get your transcript and summary files
